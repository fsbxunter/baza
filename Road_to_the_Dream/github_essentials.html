GitHub Guide

Пошаговая шпаргалка для новичков по работе с Git и GitHub

Прогресс: 0%

1
Скачайте и установите Git с официального сайта, затем настройте свои данные.
git config --global user.name "Ваше Имя"
git config --global user.email "ваш@email.com"
2
Перейдите в папку вашего проекта и инициализируйте Git репозиторий.
git init
3
Добавьте файлы в staging area (подготовительную зону) для последующего коммита.
git add .
# или для конкретного файла:
git add index.html
4
Создайте коммит с описанием изменений. Коммит - это снимок состояния вашего проекта.
git commit -m "Мое первое изменение"
5
Создайте новый репозиторий на GitHub через веб-интерфейс (без README, .gitignore и лицензии).
6
Свяжите ваш локальный репозиторий с удаленным репозиторием на GitHub.
git remote add origin https://github.com/ваш-логин/ваш-репозиторий.git
7
Отправьте ваши локальные коммиты на GitHub. Флаг -u устанавливает отслеживание ветки.
git push -u origin main
